== Sample Blog
This is a basic sample blog programmed with ruby on rails 3 and Bootstrap. It's just a persoal project to practice and improve my skills with Ruby on Rails.

== Features
1. Allows users to create a blogger account
2. Registered users can create posts
3. Everyone can comment on blog posts
